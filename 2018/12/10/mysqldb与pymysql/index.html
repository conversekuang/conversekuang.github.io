mysqldb与pymysql

发表于 2018-12-10

MySQLdb与 PyMySQL

之前一直用的python2.7版本，用到的是MySQLdb这个模块，然而又经常在资料中看到连接mysql数据中的是pymysql，于是就对这两个模块进行比较一下有什么区别。最后发现，其实也并没有多少差别。使用方法上几乎相同。

[TOC]

MySQLdb与PyMySQL的相关介绍

MySQLdb只支持py2.x版本，而pymysql支持2.7以及3.x版本

MySQLdb

  1. 目前支持MySQL版本3.23 - 5.5 并且 Python版本2.4 - 2.7
  2. 安装命令 pip install MySQL-python
  3. MySQLdb 是基于_mysql 的python简易包装器（官方文档中的用的是thin wapper，不知道这样翻译对不对），它能兼容 Python DB API interface (version 2)[PEP-0249]，所以大部分使用方式还是根据[PEP-0249]定义。只有MySQLdb模块中与接口定义不同的时候，参考MySQLdb模块文档[MySQLdb]。其实MySQLdb只是定义了一些顶层的一些函数和属性，底层的很多东西在_mysql。

_mysql

如果要编写可跨数据库移植的应用，使用 MySQLdb模块而避免直接使用_mysql模块。_mysql 是一个基本上实现MySQL C API的接口，更多内容参见 MySQL documentation。因为我们使用MySQLdb模块基本上使用的是高级别的函数。

所以第3点可以概括为，MySQLdb实现了 Python 数据库 API 规范 V2.0，基于 MySQL C API 上建立的。

PyMySQL

  1. 安装环境

    • python版本

    ​ CPython : 2.7 and >= 3.4

    ​ PyPy : Latest version

    • MySQL Server版本

    ​ MySQL >= 5.5

    ​ MariaDB >= 5.5

  2. 安装命令 pip install pymysql

  3. pymysql是基于PEP 249 的纯python MySQL客户端库。大多数的API与mysqlclient 和MySQLdb兼容。注意的是，pymysql并不支持低级别的_mysql接口。

因此，pymysql基本可以代替MySQLdb，解决因为python版本带来的问题。
